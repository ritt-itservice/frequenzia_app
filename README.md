# 🌍 Frequenzia

Eine freie, werbefreie Android-App zum Entdecken und Hören von Internetradio-Sendern
aus aller Welt.

## Features

- 🔍 **Sendersuche** nach Name, Land oder Genre/Tag
- ▶️ **Hintergrundwiedergabe** – läuft weiter bei gesperrtem Bildschirm, mit Steuerung
  über Benachrichtigung und Lockscreen
- ⭐ **Favoriten** lokal speicherbar
- 🌐 Zugriff auf über 50.000 Sender weltweit via [Radio Browser API](https://www.radio-browser.info/)
- 🔓 100 % Open Source, keine Tracker, keine Werbung, keine Cloud-Bindung

## Tech Stack

- Kotlin + Jetpack Compose
- Media3 (ExoPlayer) für Streaming & Background Playback
- Retrofit für API-Anbindung
- Room für lokale Favoriten
